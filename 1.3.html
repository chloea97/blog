1.3 Plan and carry out tasks using IT

As aforementioned, planning is the most important consideration for any digital project. There are a number of platforms that assist project planning and management each with respective advantages and disadvantages. Trello is a common management tool for small groups or individual projects.

Whilst it enables you to track individual tasks through a process, it does not allow you to assign deadlines to tasks which is a key part of time management. It is therefore key when using Trello to either use a separate tool or be extremely mindful of any time constraints within the project.
