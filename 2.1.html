1.2 Describing methods, skills and resources needed to complete digital projects successfully

Whilst the core IT tools (Visual Studio Code, GitHub and Trello) remained the same throughout the process, other tools, however were added along the way.

These included:

  • Google Fonts - Provides free fonts for commercial use and is easily integrated with CSS
  • Unsplash - Library of free for commercial use images
  • Slack - Despite the project not being within a group, Slack was used for collaboration and code sharing within the class
  • W3 Schools - An online tool for examples and tutorials for HTML and CSS that helped me to use extra functions outside of what was learnt
