1.7 Describe which are the appropriate file types for websites

When making a website there are lots of different file types that correspond to different coding languages each of which have different functions when creating a website. The most frequently used include:

  • CSS - Cascading StyleSheets take care of how websites look [7]
  • HTML - HyperText MarkUp Language [7]
  • JavaScript - Used to make websites with static HTML and CSS interactive [6]
  • JPEG - Joint Photographic Experts Group file - are image files mainly used for photos [7]
