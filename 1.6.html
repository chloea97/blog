1.6 Describe and provide analysis on how your chosen technologies have helped you achieve your outcomes

Throughout the process a number of different technologies have enabled me to achieve the final product. Each of these were selected in the planning process in order to streamline the digital project or achieve certain requirements. These include:

Trello

Trello is a task management tool that allows you to track the progress of broken down elements of the project. It helps you to gain an overview of how much you have done and how much you still have left to you. It also helps you to not forget any tasks.

Git

Git is a revision control system that manages your source code history. This helps with version control and efficiency within projects. [8]

GitHub

GitHub is a hosting service for Git repositories, it allows your to share the and storie copies, helping with collaboration and version control. [8]
