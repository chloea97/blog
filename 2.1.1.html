1.1 Describe the content and layout for each page

Whilst all of the pages follow the same basic layout, with most of the features overlapping such as tab bar, references, main image and title, as each page has a different function other features differ. As mentioned, individual pages require different content and execute different functions as they have different purposes within the user journey.The key pages on the website include; Home, About Me, Blog and Individual Blog pages.

Home page

It is considered good UX practise to include a link to have references to all of the key pages on the website on the Home page. I therefore included links to both the blog and the about me pages, before including recommendations for individual blog pages.
